## Сборка

Чтобы собрать программу, для начала склонируйте репозиторий и перейдите в директорию `curveplotter`.

```bash
git clone https://github.com/ronnie256sm/CurvePlotter.git && cd CurvePlotter/curveplotter
```
Затем, выполните сборку программы.

```bash
dotnet build -c Release
```

Собранная программа будет находится в директории `./bin/Release/net9.0/`, где будет находиться исполняемый файл `curveplotter`.

Если же вы хотите собрать Debug-версию программы, то программу надо собирать другой командой.

```bash
dotnet build -c Debug
```

Собранная программа будет находится в директории `./bin/Debug/net9.0/`.

## Описание

Данная программа предназначена для визуализации одномерных кривых на координатной плоскости, где кривые задаются явным выражением функции, а также для построения некоторых типов сплайнов. Программа поддерживает следующие виды функций:

- базовые алгебраические операции (сложение, вычитание, умножение, деление);
- линейные и степенные функции, функция квадратного корня;
- логарифмы (натуральный, десятичный, с произвольным основанием);
- экспонента;
- тригонометрические функции (синус, косинус, тангенс, котангенс, секанс, косеканс);
- обратные тригонометрические функции (арксинус, арккосинус, арктангенс, арккотангенс, арксеканс, арккосеканс);
- гиперболические функции (гиперболический синус, гиперболический косинус, гиперболический тангенс, гиперболический отангенс, гиперболический секанс, гиперболический косеканс);
- сигнум.

Также, помимо функций, поддерживаются следующие типы кривых:

- ломаная;
- интерполяционный кубический сплайн с производными, построенными с помощью полинома Лагранжа;
- интерполяционный кубический сплайн с непрерывными вторыми производными;
- сглаживающий кубический сплайн с α- и β-регуляризацией.

Программа содержит справку и подсказки, в которых описан весь функционал программы.

## Известные проблемы

- Возможны вылеты программы на операционных системах, отличных от Windows, при некорректном вводе данных сплайна;
- Сигнум отображается некорректно;
- Имеются проблемы с корректностью отображения резко возрастающих или убывающих функций, а также функций, очень близких к нулю.
